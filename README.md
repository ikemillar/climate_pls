# Climate Drivers Interaction Analysis using PLS-SEM

This repository contains R code for estimating climate
relationships using Partial Least Squares Structural Equation
Modeling (PLS-SEM).

## Research Objective

The model examines interactions between:

- Relative Humidity (RH)
- Maximum Temperature (TMAX)
- Minimum Temperature (TMIN)
- Wind Speed (WIND)
- Rainfall (RAIN)

using climatological observations from Ghana.

## Methods

The analysis uses the R package **seminr** to estimate:

1. Measurement model
2. Structural model
3. Mediation effects
4. Moderation effects
5. Predictive performance (PLS-Predict)
